Our jam entry for [Ludum Dare 48][ld].  
Download it for Windows, Linux, and MacOS on [itch.io]!  

## Controls
| Key      | Action           |
| -------: | :--------------- |
| `A/D`    | move left/right  |
| `Space`  | start            |
| `Escape` | pause            |
| `Q`      | quit             |
| `S`      | skip tutorial    |
